# Умный список задач (Smart Todo App)

## Описание проекта
Умный список задач - это современное desktop-приложение для управления задачами, разработанное на PyQt5. Приложение предоставляет удобный интерфейс для создания, редактирования и отслеживания задач с поддержкой категорий, приоритетов и сроков выполнения.

## 🚀 Функциональные возможности

### Основные функции
- ✅ **Создание, редактирование и удаление задач**
- 📂 **Организация задач по категориям** с цветовым кодированием
- 🎯 **Установка приоритетов** (Низкий, Средний, Высокий)
- 📅 **Управление сроками выполнения** с календарем
- 🔍 **Поиск и фильтрация задач** по различным критериям
- 📊 **Визуальная статистика** выполнения задач
- 🎨 **Настройка внешнего вида** (темы, шрифты)

### Дополнительные возможности
- 🔊 **Звуковые уведомления** при добавлении задач
- ⌨️ **Горячие клавиши** для быстрого доступа
- 🖱️ **Контекстное меню** для быстрых действий
- 💾 **Автосохранение** настроек
- 🌙 **Поддержка тем** (светлая, темная)

## 🛠 Технологии

- **Python 3.8+** - основной язык программирования
- **PyQt5** - для графического интерфейса
- **SQLite** - для хранения данных
- **Qt Designer** - для создания UI форм

- Python 3.8 или выше

# Горячие клавиши

Сочетание	Действие
Ctrl + N	Добавить новую задачу
Ctrl + E	Редактировать выбранную задачу
Delete	Удалить выбранную задачу
Ctrl + F	Фокус на поле поиска
Ctrl + S	Открыть настройки
F5	Обновить список задач
